15.5 Recurrent networks

  • Once we learn the general equation and properties of the line / hyperplane we can turn around and perform various learning tasks - like regression and classification - using it as a model.
  • By the same token now that we have a basic understanding of how to model general ordered data, we can use those models to perform various learning tasks.
  • In this Section we use the generic variable order dynamic system introduced in the previously to perform simple learning tasks on time series. This is often referred to as Recurrent Neural Networks (or RNNs for short). RNNs can also be used for more sophisticated time series problems - like machine translation - which we discuss in the next Section.

15.6.1 Simple time series prediction¶

  • Recurrent Neural Networks (RNNs) = using a variable order dynamic system as model for learning
  • Their value over fixed order systems (autoregressive processes) lies in the more complete memory of variable order systems (see Section 15.3 for a discussion of this topic)
  • When we use them for prediction tasks we hope our hidden states learn interesting and useful summary statistics - of our entire dataset - that we can fruitfully leverage
  • The simplest variable order system we can use:
\begin{equation} h_{t+1} = w_0 + w_1h_{t} + w_2x_{t+1} \end{equation}
  • Note however, for prediction we want to use this to predict $x_{t+1}$, so we need to kick back the index on $x$ by one and use
\begin{equation} h_{t+1} = w_0 + w_1h_{t} + w_2x_{t} \end{equation}
  • Now to perform simple time series prediction we employ this as our model, and use an appropriate cost (and training data of course) to tune its parameters appropriately

def step(h_t_prev,x_t_prev,w):
    return w[0] + w[1]*h_t_prev + w[2]*x_t_prev
    
# exponential average function
def model(x,w):
    # set initial conditions of h to values of x
    h = [x[:,0]]
    
    # range over x and create h
    for t in range(0,np.size(x)-1):
        # get current point and prior hidden state
        h_t_prev = h[-1]
        x_t_prev = x[:,t]

        # make next element and store
        h_t = step(h_t_prev,x_t_prev,w)
        h.append(h_t)
    return np.array(h).T

# an implementation of the least squares cost function for linear regression
def least_squares(w,x,y):
    # compute cost over batch
    cost = np.sum((model(x,w) - y)**2)
    return cost/float(np.size(y))
